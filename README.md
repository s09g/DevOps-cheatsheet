# DevOps-cheatsheet

初稿是2022年我为谷歌云客户提供的教程。讲稿修改成了专题讲义开源分享。

我在谷歌的时候参与了kubernetes上负载均衡和IPv6的开发。最初的两年还挺开心，后来就觉得不对劲，越干越像devops，技术路线走窄了。

于是开始调整技术方向，离开谷歌加入Meta开始做机器学习。

想和我一起研究系统设计和机器学习系统，可以加入知识星球一起学习。
